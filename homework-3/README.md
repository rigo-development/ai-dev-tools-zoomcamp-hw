# MCP Server Clone

This is a clone of Context7 built as an MCP server using `FastMCP`.

## Features

- [ ] Select a GitHub repo with documentation
- [ ] Download data from the repo
- [ ] Make the data searchable

## Getting Started

### Prerequisites

- [uv](https://docs.astral.sh/uv/)

### Installation

```bash
uv init
uv add fastmcp
```

### Running the Server

```bash
uv run main.py
```

## Usage

This server provides tools and resources to interact with documentation from GitHub repositories.
